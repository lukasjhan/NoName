#BIOS color attributes

##Colors

* 0x00 : BLACK
* 0x01 : BLUE
* 0x02 : GREEN
* 0x03 : CYAN
* 0x04 : RED
* 0x05 : MAGENTA
* 0x06 : BROWN
* 0x07 : LIGHT GREY
* 0x08 : DARK GREY
* 0x09 : LIGHT BLUE
* 0x0A : LIGHT GREEN
* 0x0B : LIGHT CYAN
* 0x0C : LIGHT RED
* 0x0D : LIGHT MAGENTA
* 0x0E : YELLOW
* 0x0F : WHITE
